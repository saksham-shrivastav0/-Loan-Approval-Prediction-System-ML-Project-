# 🏦 Loan Approval Prediction System (ML Project)

## 📌 Overview

This is my **first Machine Learning project** where I built a complete pipeline to predict whether a loan will be approved or not based on applicant details.

The project covers **data cleaning, preprocessing, feature engineering, model building, and evaluation** using multiple ML algorithms.

---

## 🚀 Features

* Real-world dataset handling (uncleaned data)
* Data cleaning & preprocessing
* Handling missing values
* Feature scaling (without data leakage)
* Encoding categorical variables properly
* Multiple ML models:

  * Logistic Regression
  * K-Nearest Neighbors (KNN)
  * Decision Tree
* User input-based prediction system
* Model comparison

---

## 📊 Dataset

* Loan Prediction Dataset (real-world style)
* Contains features like:

  * Applicant Income
  * Coapplicant Income
  * Loan Amount
  * Loan Term
  * Credit History
  * Property Area
  * Marital Status
  * Employment Status

---

## 🧠 ML Pipeline

1. Data Loading
2. Data Cleaning
3. Handling Missing Values
4. Encoding Categorical Data
5. Feature Selection
6. Train-Test Split
7. Feature Scaling (StandardScaler)
8. Model Training
9. Prediction
10. Model Evaluation

---

## ⚙️ Models Used

### 1. Logistic Regression

* Used for binary classification
* Fast and interpretable

### 2. K-Nearest Neighbors (KNN)

* Distance-based model
* Sensitive to scaling

### 3. Decision Tree

* Non-linear model
* Does not require scaling

---

## 📈 Evaluation Metrics

* Accuracy Score
* (Initially tried MSE/RMSE but later corrected to proper classification metrics)

## 💻 User Input Prediction

The system takes user inputs such as:

And predicts:
👉 **Loan Approved / Rejected**
